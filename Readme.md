# ISYS-2099 Database Application - Database Project - Group 4

## Project Structure

## Technology Stack

- Database: MySQL and MongoDB.
- Backend API Server: NodeJS, Express, and Mongoose.
- Frontend: React, Tailwind CSS, and Axios.
